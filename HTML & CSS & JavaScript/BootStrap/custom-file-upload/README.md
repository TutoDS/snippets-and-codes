# Custom File Upload

-   **Files:**

    -   `custom-file-upload.css`

I save this code because I want customize a **_"custom file input"_** on BootStrap.
With this code the input appear this way:

-   **1 file**: appear file name
-   **Multiple files**: appear count of files
